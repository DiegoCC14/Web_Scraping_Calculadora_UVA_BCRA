	Esta aplicacion, mal llamada UVAS, ya ahora tambien calcula 
dolares, esta app de escritorio, le servira para saber:
	
	- El precio del Dolar
	- El precio del UVA
	- Cuantos Pesos tiene en UVAS
	- Cuantos Pesos tiene en Dolares Blue
	- La hora actual(chiste :) )

	Para su uso solo hace falta dirigirse a la carpeta dist, donde encontrara
un ejecutable con la app funcional, con el nombre View_Principal.exe, 
	
	Si su idea es modificar el codigo, entonces debera instalar Python3 y dependencias
que se encuentran en librerias_usadas.txt y ejecutar View_Principal el cual llamara a las 
clases Pestania_UVA.py u Pestania_Dolar.py en donde se encuentra la logica de cada Pestania
